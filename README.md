# Java Bank Exception Handling Example

This Java program demonstrates the use of:

• User-defined Exceptions  
• Built-in Exceptions  
• Try-Catch-Finally blocks

## Concepts Used
- Custom Exception (InsufficientFundsException)
- NumberFormatException
- Exception Handling
- Bank withdrawal simulation

## Program Flow
1. User enters initial bank balance
2. User enters withdrawal amount
3. Program checks if balance is sufficient
4. If insufficient → user-defined exception is thrown
5. Finally block prints transaction status
